# DSA-CPP

A structured collection of Data Structures & Algorithms practice programs written in **C++**, organized chapter-wise as I work through core CS concepts.

## 📖 About

This repository is a growing series documenting my journey through data structures, algorithms, and foundational computer science topics. Each chapter focuses on a specific concept, with hands-on C++ implementations, multiple problem variations, and (where useful) more than one approach to the same problem for comparison.

> More topics will be added as I progress through the course/curriculum.

## 🛠️ Tech Stack

- **Language:** C++
- **Compiler:** G++ / MinGW

## ▶️ How to Run

Navigate into any folder and compile the desired file:

```bash
g++ filename.cpp -o filename
./filename
```

## 🎯 Goal

To build a solid, well-documented foundation in data structures and algorithms while creating a public record of progress — useful both for revision and as a portfolio reference.
